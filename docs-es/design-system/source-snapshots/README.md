# Source Snapshots

Esta carpeta guarda referencias importadas del design-system actual de Android e iOS.

Objetivo:

- Preservar trazabilidad respecto al estado de origen.
- Facilitar decisiones de migracion y limpieza de naming.
- Reducir coste de redescubrimiento durante implementacion.

Archivos:

- `current-android-design-system.md`
- `current-ios-design-system.md`
- `current-design-system-files-inventory.md`

Notas:

- Estos snapshots son entrada de referencia, no reglas inmutables.
- Las reglas canonicas viven en los documentos padres (`foundations`, `components`, `governance`).
