Reglas de diseño

Movimiento

El menu no aparece encima del Home. El Home se traslada horizontalmente, conserva su contenido a la derecha y proyecta sombra sobre el drawer.

Identidad

Mostrar foto de perfil o familia cuando exista. Si no hay imagen, usar el logo circular de La Reguerta como fallback.

Secciones

Primero funciones visibles para todos. Despues funciones de productores. Finalmente herramientas admin, solo si el usuario tiene permisos.

Version

La version siempre queda anclada abajo. En develop se añade una etiqueta DEV en color de aviso para distinguirla de produccion.
